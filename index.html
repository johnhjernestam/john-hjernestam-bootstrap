Element

Element har i de flesta fall en start- och sluttag. Webbsidor bygger på element
som utgör själva innehållet på webbsidan. Exempel på element är div, p och h1.

HTML

HTML är det vanligaste datorspråket för att skapa webbsidor. HTML utgör strukturen för en webbsida och CSS designar.
Strukturen kan innehålla allt från rubriker till bilder som du sedan med CSS kan positionera och designa.

ID

ID:et är något du döper ett objekt eller område till för att kunna
designa med CSS. Man använder ID för att specifiera något och
det får därför bara användas en gång i ett HTML-dokument.

Class

Class är något du döper ett objekt till för att kunna designa med CSS. Skillnaden mellan class
och ID är att flera element kan tillhöra en och samma klass medan ID bara kan användas en gång.

Meta

Meta är inte synligt på ens webbsida men är viktigt eftersom man anger information om sidan till exempel vem som skapat den,
vad den handlar om och sökord. UTF-8 är en typ av kodning som man kan lägga i en meta-tag vilket stödjer
många språk och gör det möjligt för oss svenskar att kunna se bokstäverna å,ä, och ö på en webbsida.

Responsive

När man pratar om responsive pratar man oftast om en webbsida är mobilanpassad. Om en webbsida är
responsive så är den funktionell och ser bra ut oavsett om skärmen är stor eller liten.

http

Http står för Hypertext Transfer Protocol och är ett kommunikationsprotokoll som
gör det möjligt för en webbläsare att kommunicera med en webbsida.

https

Https står för Hypertext Transfer Protocol Secure och är ett kommunikationsprotokoll som gör det möjligt för
en webbläsare att kommunicera med en webbsida. Skillnaden mellan http och https är att https-data
som skickas är krypterad vilket gör det mycket säkrare för användare att surfa på internet.

Compressed Data

Compressed data eller på svenska datakompression är när man tar data och kodar om
det på ett annat sätt som gör att man behöver använda färre informationsbärande enheter.

CDN

CDN är ett distributionsnätverk som lagrar information och gör det snabbare att komma in på webbsidor. Om man
har webbsiodor som behöver nås av många människor över hela världen är det viktigt att ha en CDN-server. Det minskar drastiskt
tiden det tar att komma in på webbsidan för användare som bor på andra sidan jordklotet.

Local Bootstrap

Local Bootstrap är ett biblotek som används som en grund/ram när man skapar webbsidor. Det innebär att om du bara skriver
din HTML-kod med element och klasser som delar samma namn som används i bootstrap-filen så kommer allt redan
vara designat (bootstrapens CSS). Man kan själv göra en egen CSS-fil och skriva över koden som finns i Bootstrap.

JQuery

JQuery är världens mest använda Javascript-bibliotek som gör det
enklare att kommunicera och förändra element på en webbsida.

Flexbox

Flexbox gör det möjligt för element i containrar att ordnas automatiskt beroende på hur stor skärmen är.
Det är mycket användbart för att göra mobilanpassade webbsidor då skärmstorleken varierar.

Grid

Grid-layouter ser olika ut men har det gemensamt att de är indelade i rutor. Det förenklar för en eftersom man kan välja en
layout som passar för ens hemsida och därmed börja jobba direkt utan att behöva positionera rubriker, menyer, footer osv.

Code snippets

Code snippets är oftast små områden av kod (eller text mm.) som kan återanvändas på andra ställen i ens
HTML-dokument. Det är mycket användbart att spara kod som återkommer regelbundet i
ens dokument. Det kallas för code snippets/utdrag avkod och man kan
spara dem under "utdrag" i de allra flesta code editors.
